This game was made for the Bigmode Game Jam 2026.



DOPANIUM is a top-down shooter where hacking is at the core of gameplay. Don't shoot on the robots: make them shoot themselves. Use their own systems against them to wipe the zone in a slick shootout. But beware : you can only hack for so long before the connection cuts.



Gameplay:

The goal is simple : eliminate all robots to survive.

Use WASD to locate enemy robots. When you see one, hover over it and press left click to take control of it.

Once in hack mode, you can move the enemy with WASD. You have two hacks at your disposal: shoot (with left click) and self-destruct (with right click). When you are done hacking the robot, press [space] to get back to the selection.

But beware! Each level has a set time limit .When hacking, the timer counts down. If it reaches zero, the connection is cut and all the instructions are executed. Moreover, you have a set number of shots (red usb) and self-destructs (yellow usb). Use them wisely!

Once you are done hacking the robots, press [space] to execute all instructions simulaneously. You can now move, so try not to get shot by the robots you controlled!
